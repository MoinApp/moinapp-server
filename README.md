# MoinApp Server

The server for Moin.

**Moin** is an Instant Messenger based on one word: Moin.

The server is deployed to http://moinapp.herokuapp.com/ (this link is funny).

## Requirements

- node.js >= v0.10.x
- npm >= 1.5.x
